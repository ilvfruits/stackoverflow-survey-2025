## Dataset

This analysis uses the official Stack Overflow Developer Survey 2025
(~80k respondents across 170+ countries).

## Key Question

Which programming languages are growing the fastest?

## Results

![Language Popularity](/assets/images/language_popularity.png)

### Observations

- Python continues steady growth
- Rust has strong interest-to-usage ratio
- SQL remains universal but not “aspirational”

## Methodology

- Cleaned multi-select responses
- Normalized counts by respondent
- Excluded missing values
